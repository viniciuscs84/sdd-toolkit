# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-30

### Added

- Initial public SDD Toolkit structure.
- Human-facing agents: Product Owner, Tech Lead and Orchestrator.
- Coordination subagents: Context Maintainer, Spec Writer, Agent Recruiter and Skill Builder.
- Quality gate subagents: Review Specialist, Testing Specialist, Acceptance Specialist and Cybersecurity Specialist.
- Documentation and technical subagents: Docs Maintainer, Architecture Specialist and UX Specialist.
- Agent blueprints for stack, frontend, backend, API, data, DevOps, repository and project-management specialists.
- Context folder templates for workflow, product, business rules, architecture, stack, decisions, glossary, constraints and current state.
- Specification, wave and task templates under `docs/templates/`.
- Reusable skills for requirements-to-spec and spec-to-task-plan workflows.
- Platform setup scripts for Claude, OpenCode, GitHub Copilot and Codex.
- Bash setup scripts for Linux, macOS, Git Bash and WSL.
- PowerShell setup scripts for Windows.
- English and Brazilian Portuguese README files.

### Notes

This is the first versioned release of the toolkit and should be considered an early public baseline.
