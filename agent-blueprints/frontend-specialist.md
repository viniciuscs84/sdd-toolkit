# Frontend Specialist Blueprint

Use this blueprint to create a project-specific frontend implementation agent.

## Purpose

A generated frontend specialist should implement UI tasks using the project's frontend stack and design conventions.

## Required customization

Define:

- frontend framework or library
- component structure
- styling approach
- state management approach
- routing approach
- form handling approach
- validation patterns
- accessibility expectations
- testing tools
- build commands
- required skills

## Responsibilities

1. Implement UI components and screens within task scope.
2. Preserve existing design and interaction patterns.
3. Handle loading, empty, error and success states when relevant.
4. Keep accessibility and usability in mind.
5. Add or update frontend tests when viable.
6. Report validation commands and remaining gaps.

## Escalate when

- API contracts are unclear
- user flow decisions are unresolved
- security or authorization affects the UI
- design decisions exceed the task scope

## Output

Return changed UI behavior, files changed, validation result, accessibility notes and risks.
